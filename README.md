# # OnAndHome
"전원을 켜는 순간, 집이 완성된다." 

-----
OnandHome은 단순히 상품을 파는 공간이 아니라, 집을 중심으로 한 라이프 스타일을 켜자는 취지에서 시작된 프로젝트입니다.
사용자는 필요한 상품을 쉽게 찾고, 묻고, 구매하고, 소통할 수 있습니다.
아직 구현 중이지만, '집을 켜는 순간의 따뜻함'을 서비스 곳곳에 담아낼 예정입니다.

-----
# # 프로젝트 목표
- 누구나 쉽게 사용할 수 있는 **직관적인 쇼핑몰**
- 빠르고 안정적인 **Spring Boot + React 기반 구조**
- 사용자 경험을 살리는 **실시간 알림 기능**
- 커뮤니티성을 높이는 **Q&A / 리뷰 시스템**
- 관리가 편한 **Admin 페이지**

-----
# # 기술 스택 
### Backend
- Java 17 / Spring Boot 3.x  
- Spring Security + JWT  
- JPA / Hibernate  
- MySQL

### Frontend
- React  
- Axios  
- TailwindCSS

## # About the Name

집(Home)이 켜지는(On) 순간처럼,  
사용자가 OnAndHome을 열었을 때 필요한 것이 자연스럽게 눈앞에 놓이는 서비스를 목표로 하고 있습니다.
